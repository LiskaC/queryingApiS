# Querying APIs

## Getting started

### 1. Clone the project

```bash
git clone <paste the repo URL here>
cd queryingApiS
```

Then, open two terminal windows — one for the backend, one for the frontend:

### 2. Install dependencies

You need to install separately for the backend and the frontend.

**Terminal 1 — Backend:**

```bash
cd backend
npm install
```

**Terminal 2 — Frontend:**

```bash
cd frontend
npm install
```

### 3. Run the project

**Terminal 1 — Backend:**
Run:

```bash
cd backend
npm run dev
```

The backend runs at `http://localhost:3000`

**Terminal 2 — Frontend:**
Run:

```bash
cd frontend
npm run dev
```

Then open the URL shown in the terminal (usually `http://localhost:5173`) in your browser.
